# Django Project

A simple Django web application built as an academic project to learn backend development, URL routing, views, and template rendering in Django.

## Features

- Basic Django project structure with `manage.py` for administrative commands.
- Configurable settings module (`pro.settings`) for environment-specific configuration.
- Support for creating apps, models, views, and URL routes inside the project.
- Ready to run using the Django development server on localhost.

## Tech Stack

- Python 3.x
- Django (latest compatible 3.x or 4.x)
- SQLite (default Django database)
- HTML, CSS (for templates)

## Project Structure

- `manage.py` – Command-line utility to runserver, migrate, create apps, etc.
- `pro/` – Main project package containing settings, URLs, WSGI/ASGI configuration.
- `apps/` (if present) – Individual Django apps for different modules.
- `templates/` – HTML templates for rendering views.
- `static/` – Static files like CSS, JS, images.
